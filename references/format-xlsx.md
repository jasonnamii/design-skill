# Format: XLSX (openpyxl)

## 폰트

Pretendard → fallback 맑은 고딕. **11pt 통일.** 1종만.

## 헤더 행

- Bold
- 배경: #f5f5f7
- 하단 테두리: thin, color #d2d2d7
- 정렬: 중앙

## 데이터 행

- Regular
- 테두리: 없음
- 행 교대 배경: #fff / #fafafa
- 정렬: 숫자=우측, 텍스트=좌측

## 여백 번역 (C4)

- 열 너비: 기본 × 1.3 (넉넉하게)
- 행 높이: 기본 × 1.2
- 셀 병합: 최소 (제목 행에서만 허용)

## 유채색

조건부서식 등에서 CTA 1색만 사용 (기본 #0066cc 또는 override).
빨강/초록 신호등 배색 지양. 대신 CTA색 농도 차이로.

## 시트 레이아웃

- 시트 탭 이름: 짧고 명확
- 첫 행 고정 (freeze panes)
- 인쇄 영역 설정 권장

## 체크리스트

- [ ] 폰트 1종 11pt 통일
- [ ] 헤더 Bold + #f5f5f7 배경
- [ ] 행 교대 배경
- [ ] 열너비 넉넉하게 (×1.3)
- [ ] 유채색 1색만
