# Special Features — S1~S18 포맷별 구현

각 S 패턴의 포맷별 구현 방법. 자동 조건 충족시 또는 형이 수동 트리거시 적용.

## "해당 없음" Fallback 규칙

포맷에서 특정 S패턴이 구현 불가("해당 없음")할 때:
1. 해당 콘텐츠의 **핵심 메시지를 텍스트로 추출**
2. 포맷의 **기본 블록(표/텍스트)으로 대체** 출력
3. 패턴의 시각적 효과는 포기하되, **정보 손실은 0**으로

| 패턴 | XLSX fallback 예시 |
|------|-------------------|
| S4 아이콘 그리드 | 열1=항목명, 열2=설명 (데이터 구조로 대체) |
| S7 감성 선언 | 병합셀 + Bold + 전후 빈 행 3개로 여백 확보 |
| S8 좌우분할 | 열1=문제, 열2=해결 (2열 표로 대체) |
| S15 디바이스 목업 | 이미지 링크 텍스트로 대체 |

## S1. 빅넘버

숫자가 블록 핵심 메시지일 때. 금액·시간·성능·%.

| 포맷 | 구현 |
|------|------|
| HTML | 숫자: L1급 font-size + wght 900. 단위: L3급. 설명: L4. 숫자 color=CTA or 흰(다크) |
| MD | `# 숫자` (H1) + 본문 설명. 숫자 행 단독 |
| PPTX | 숫자: 44pt+ wght 900 중앙. 단위/설명: 18pt 하단 |
| DOCX | 숫자: 28pt Bold 단독 행. 설명: 11pt 하단 |
| XLSX | 숫자 셀: Bold + 크기 유지(11pt). 조건부서식 CTA색 배경 |
| PDF | 숫자: L1 pt + wght Black. 단위: L3 |

## S9. 형광펜 (하이라이트)

3줄+ 텍스트에서 핵심 구절 강조. G7: 1블록 2구절 이내.

| 포맷 | 구현 |
|------|------|
| HTML | `<mark>` + `background: linear-gradient(transparent 60%, var(--cta, #0066cc)33 60%)` 밑줄형 형광 |
| MD (옵시디언) | `==핵심 구절==` |
| MD (일반) | `<mark>핵심 구절</mark>` |
| PPTX | 해당 텍스트 run의 font.highlight_color = CTA색 밝은 버전 |
| DOCX | 해당 run의 font.highlight_color. WD_COLOR_INDEX 중 YELLOW 또는 커스텀 |
| XLSX | 해당 셀 배경 CTA색 10% 투명도 |
| PDF | 텍스트 뒤 rect(CTA색, alpha=0.2) 배치 |
