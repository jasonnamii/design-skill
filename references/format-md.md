# Format: Markdown

## 타이포 위계

| 레벨 | 문법 | 용도 |
|------|------|------|
| L1 | `# 제목` (H1) | 문서 제목, 히어로 |
| L2 | `## 제목` (H2) | 섹션 제목 |
| L3 | `### 제목` (H3) | 소제목, 블록 제목 |
| L4 | 본문 | 설명, 상세 |
| XS | *이탤릭* 또는 작은 텍스트 | 캡션, 각주 |

## 여백 (빈 줄로 번역)

| 구간 | 빈 줄 수 |
|------|---------|
| 섹션(##) 간 | 3줄 |
| 블록 간 (같은 섹션 내) | 2줄 |
| 단락 간 | 1줄 |

## 강조 규칙

- **Bold**: 제목과 핵심 키워드에만. 본문에서 Bold 남발 금지 (1블록 1~2개 이내)
- *이탤릭*: 캡션, 출처, 부가정보에만
- ~~취소선~~: 사용 금지
- 밑줄: 사용 금지

## 콜아웃 체계

콜아웃 사용 규칙은 **`references/protocol-pretty.md` (이쁘니 프로토콜)이 SSOT**. 이쁘니의 8종 팔레트를 따른다. 용도별 다종 사용 허용 (남용 금지 규칙은 이쁘니에 명시).

## 정렬

md에서는 전부 좌측정렬 (C5 중앙정렬 면제). 표 내부만 정렬 지정 가능.

## 구분

`---` (hr) = 최상위 섹션 구분에서만. 나머지는 빈 줄로.

## 하이라이트

- 옵시디언: `==핵심 구절==`
- GitHub/일반: `<mark>핵심 구절</mark>`

## ⛔ 인라인 HTML 금지 (`.md` 전용 강제)

**순수 마크다운 DEFAULT.** `.md` 산출 시 인라인 HTML **전면 금지**.

| 금지 | 이유 |
|---|---|
| `<div>`·`<span>` 래퍼 | 옵시디언 Reading view·GitHub 렌더 깨짐 빈번 |
| `style="..."` 속성 | border·background·gradient 옵시디언 테마 충돌 |
| `<p>`·`<br>` 수동 태그 | md 기본 문법(빈줄·2space)이 상위 |
| CSS 클래스·ID | md 렌더러 무시 |

**허용:** 헤더(#)·불릿(-)·번호(1.)·표(|)·인용(>)·코드블록(```)·이모지·옵시디언 하이라이트(==)·콜아웃(`> [!note]`)

**예외 (사용자 명시시만):**
- "HTML로" → 풀 HTML 전환
- "박스로"·"카드로" → html-div-style cascade
- "벤토로"·"시각화" → apple-box-design cascade

**검증:** `grep -E "<div|<span|style=" output.md` = ∅ 필수. ≠∅ 시 FAIL → 재작성.
