# Tone: clean-info

스펙·비교·가격·데이터·가이드. Apple "비교하기" "기술 사양" "구입하기" 페이지 감성.

## 배경

흰(#fff) + 밝은그레이(#f5f5f7) 교대. 다크 섹션 0~1개.
정보 밀도가 높으므로 배경 전환으로 영역 구분.

### CORE 예외 (§1 CORE 예외 규정에 따라)

**C1 조정**: L1:L2 비율 4:2.3 → 3:2.3.
- **공리 근거**: X2-a (고밀도 정보에서는 위계 간격 축소가 탐색 효율을 높인다)
- **조정 범위**: L1:L2 간격만. L3:L4 비율 불변.

## 이미지

- 제품 컷아웃(배경 제거) 위주
- 비교시 동일 크기·동일 앵글로 정렬
- 이미지 역할: 설명 보조 (장식 아님)
